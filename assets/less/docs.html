Animation

path: /inc/animation.less

.bkg-color

Default properties

@s360--name : s360-bkg-color;
@s360--from : rgba(0,0,0, 0.8);
@s360--to   : rgba(0,0,0, 0);

How to use

Define keyframes animation with default properties.

#animation.bkg-color.new();

output:

@keyframes s360-bkg-color {
    from {
        background-color: rgba(0, 0, 0, 0.8);
    }
    to {
        background-color: rgba(0, 0, 0, 0);
    }
}

Define keyframes animation with custom properties.

#animation.bkg-color.new({
    .callback(){
        @s360--name : bkg-fade-out;
        @s360--from : rgba( 0,0,0, 1);
    }
});

output:

@keyframes bkg-fade-out {
    from {
        background-color: #000000;
    }
    to {
        background-color: rgba(0, 0, 0, 0);
    }
}

.nav-mobile

Default properties

@s360--name             : s360-nav-mobile;

@s360--from-left        : -240px;
@s360--from-box-shadow  : 0px 0px 20px 10px rgba( 0,0,0, 0.0 );

@s360--to-left          : 0px;
@s360--to-box-shadow    : 0px 0px 20px 10px rgba( 0,0,0, 0.15 );

How to use

Define keyframes animation with default properties.

#animation.nav-mobile.new();

output:

@keyframes s360-nav-mobile {
    from {
        left: -240px;
        box-shadow: 0px 0px 20px 10px rgba(0, 0, 0, 0);
    }
    to {
        left: 0px;
        box-shadow: 0px 0px 20px 10px rgba(0, 0, 0, 0.15);
    }
}

Define keyframes animation with custom properties.

#animation.nav-mobile.new({
    .callback(){
        @s360--name         : mobile-slide;
        @s360--from-left    : -350px;
    }
});

output:

@keyframes mobile-slide {
    from {
        left: -350px;
        box-shadow: 0px 0px 20px 10px rgba(0, 0, 0, 0);
    }
    to {
        left: 0px;
        box-shadow: 0px 0px 20px 10px rgba(0, 0, 0, 0.15);
    }
}
